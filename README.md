# security-learning-portfolio

This repository documents my hands-on cybersecurity learning journey with a focus on:

- Security operations and detection fundamentals
- Web and application security concepts
- Ethical offensive security labs and exercises
- Clear, responsible technical documentation

All content is based on legal training platforms (TryHackMe, Hack The Box Academy, labs, and coursework).

## Structure

- `/tryhackme` — THM room notes and summaries
- `/hackthebox` — HTB Academy modules and learning takeaways
- `/owasp` — OWASP Top 10 write-ups and examples
- `/notes` — Study notes, checklists, and reference material

## Disclaimer

All activities documented here are performed in controlled, authorized environments for educational purposes only.
